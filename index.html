Slatify

Minimal Markdown to HTML generator for beautiful documentations.

Usage

  1. Install package
     $ npm install -D slatify
  2. Add run command to package.json
     {
   ...
   "scripts": {
     "build-html": "node node_modules/lib/slatify-cli README.md index.html"
   },
   ...
 }

  3. Build html

     $ npm run build-html
